# 信息源甄别与交叉验证指南

## 一、可信信息源分级

### Tier 1：权威专业源（最高可信度）
- 国家市场监督管理总局、中国消费者协会等官方机构的检测报告
- 知名第三方测评实验室的实测数据（如中消协比较试验）
- 国际权威评测机构：Consumer Reports、Which?、Stiftung Warentest
- 垂直领域的专业媒体深度评测（需判断是否为付费软文）

### Tier 2：社区真实源（高可信度，需筛选）
- 什么值得买：深度晒单、长文评测、真实购买记录（警惕"首发""众测"等商业合作内容）
- 知乎：高赞专业回答，关注答主历史内容判断是否带货（近期集中推同一品类=嫌疑）
- 小红书：真实用户晒单（看评论区互动、图片真实性；警惕模板化文案）
- B站/YouTube：实测视频（关注是否有"赞助""合作"声明）
- 电商平台的追评（尤其是带图追评，时间跨度大的更真实）

### Tier 3：平台数据源（中等可信度，用于比价和基础参数）
- 京东、天猫、拼多多、抖音电商等平台的商品详情页参数
- 品牌官网的技术规格和售后政策
- 各平台的销量排名、好评率（注意刷单痕迹）

### 应避免的信息源
- 纯带货直播间脚本、微商朋友圈
- 无任何实测数据、全是形容词堆砌的"种草"文章
- 同一时期大量账号集中推荐的同一款产品
- 只讲优点、完全不提缺点的评测

## 二、软文识别 checklist

判断内容是否为软文/广告时，检查以下信号：

1. **利益相关声明**：是否标注"赞助""合作""广告""赠测"？未声明但明显偏向=高风险
2. **语言特征**：是否充斥"绝了""封神""天花板""必入"等极端形容词？
3. **内容结构**：是否前90%讲痛点，最后10%突然推一个产品并附购买链接？
4. **账号画像**：该账号近期是否密集推荐同一品类？是否每条都带链接？
5. **评论区**：是否大量模板化好评（如"已下单""种草了"）？负面评论是否被删除或作者回避？
6. **信息密度**：是否有具体的参数对比、实测数据、使用场景细节？还是只有主观感受？
7. **跨平台一致性**：同一产品在不同平台是否都被同一批KOL推荐？

**原则**：含2个以上软文信号的内容，仅可作为辅助参考，不可作为核心决策依据。

## 三、交叉验证策略

### 3.1 多源共识验证
- 同一款产品/品牌，至少在2个以上独立来源获得正面评价，才纳入候选池
- 优先采信指出具体优缺点（而非全好评）的内容
- 关注负面评价：如果多个独立来源都提到同一个缺点，该缺点大概率真实存在

### 3.2 参数交叉验证
- 商品核心参数（功率、容量、材质等）必须在品牌官网+至少1个电商平台详情页交叉确认
- 对"独家技术""专利配方"等宣传词，搜索专利号或技术原理进行验证

### 3.3 价格交叉验证
- 同一商品的价格必须在至少2个不同平台确认
- 历史价格需通过比价工具或历史快照确认，避免"先涨后降"假优惠

### 3.4 时间维度验证
- 关注评测/评价的发布时间，排除过时信息
- 对电子产品，优先参考近6个月内的评测；对日用品，1年内的信息均可参考
- 查看商品的上市时间，警惕清仓老款冒充"新品推荐"

## 四、特殊品类注意事项

### 酒店/旅游类
- 可信源：Booking、TripAdvisor、携程/飞猪的真实用户点评（带图+详细描述）
- 警惕：过度美化的探店视频、无入住记录的"体验分享"
- 交叉验证：同一酒店在多个平台（携程、飞猪、Booking、美团）的评分和评论趋势

### 食品/保健品类
- 可信源：国家食品安全抽检信息、有资质实验室的检测报告
- 警惕："神医""祖传秘方""纯天然无副作用"等宣传
- 交叉验证：生产许可证编号、配料表、营养成分表的官方数据

### 数码/家电类
- 可信源：专业媒体的实测数据（续航、跑分、温控等）
- 警惕：只讲外观和"手感"，不提性能参数的评测
- 交叉验证：核心参数在官网+3C认证信息+专业评测中一致
